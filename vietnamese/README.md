# WCA Regulations (Vietnamese version)

This directory contains the Vietnamese translation of the official [World Cube Association Competition Regulations](https://worldcubeassociation.org/regulations/) repository.

## Credits

The following people have contributed to the translation throughout the years:

* [Nguyen Dao Quoc Anh](https://www.worldcubeassociation.org/persons/2016ANHN02) (contributed in 2023)
* [Nguyễn Hải Dương](https://www.worldcubeassociation.org/persons/2018DUON07) (contributed in 2020, 2024, 2025 and 2026)
* [Vương Thiện Trung](https://www.worldcubeassociation.org/persons/2014TRUN01) (contributed in 2018)

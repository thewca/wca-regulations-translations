WCA Regulations (Chinese version)
================================

This git repository contains the Chinese translation of the [WCA Regulations](http://worldcubeassociation.org/regulations/).

The translation is being maintained by [Yunqi Ouyang (欧阳韵奇)](https://www.worldcubeassociation.org/persons/2007YUNQ01) (contact oyyq99999@gmail.com).

Original translation by [Shuang Chen (陈霜)](https://www.worldcubeassociation.org/persons/2008CHEN27).
